Je suis offreur

Comment offrir un trajet solidaire ? Inscrivez-vous sur Ehop Solidaires via le formulaire d’inscription.

Une fois inscrit, vous vous engagez à :

  1. Répondre dans les meilleurs délais à une demande de trajet adressée par l’équipe Ehop Solidaires ;
  2. Covoiturer le demandeur pour la courte période définie au préalable ;
  3. Se mettre d’accord avec le demandeur sur le partage des frais (nous préconisons 0,12€/km) ;
  4. Mettre vos informations personnelles régulièrement à jour sur le site Ehop Solidaires.

Cette action ne vous engage qu'à covoiturer une personne de façon très ponctuelle et temporaire. A chaque demande vous restez libre d'accepter ou pas. L'équipe Ehop Solidaires se charge du reste (vous contacter, vous mettre en relation, etc.). Alors merci d'avance !

Je suis demandeur

Comment rechercher un trajet solidaire :

  1. Contactez-nous au 02.99.35.10.77 ou à l’adresse contact@ehop-covoiturage.fr ;
  2. Expliquez-nous votre besoin de trajet (départ, arrivée, date et horaires) ;
  3. L’équipe Ehop Solidaires vous recontacte dans les meilleurs délais pour vous proposer un covoiturage ;
  4. Mettez-vous d’accord avec le conducteur sur le partage des frais (nous préconisons 0,12€/km) ;
  5. Laissez-vous conduire !
